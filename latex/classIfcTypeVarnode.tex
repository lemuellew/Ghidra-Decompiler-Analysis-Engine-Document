\doxysubsection{Detailed Description}
Attach a typed symbol to a specific \mbox{\hyperlink{classVarnode}{Varnode}}\+: {\ttfamily type varnode $<$varnode$>$ $<$typedeclaration$>$} 

A new local symbol is created for the {\itshape current} function, and is attached to the specified \mbox{\hyperlink{classVarnode}{Varnode}}. The {\itshape current} function must be decompiled again to see the effects. The new symbol is {\itshape type-\/locked} with the data-\/type specified in the type declaration. If a name is specified in the declaration, the symbol is {\itshape name-\/locked} as well.
