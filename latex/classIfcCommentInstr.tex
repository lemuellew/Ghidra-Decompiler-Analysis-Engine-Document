\doxysubsection{Detailed Description}
Attach a comment to an address\+: {\ttfamily comment $<$address$>$ comment text...} 

Add a comment to the database, suitable for integration into decompiler output for the {\itshape current} function. The command-\/line takes the address of the machine instruction which the comment will be attached to and is followed by the text of the comment.
