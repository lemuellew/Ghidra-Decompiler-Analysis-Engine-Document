\doxysubsection{Detailed Description}
Create an external ref symbol {\ttfamily map externalref $<$address$>$ $<$refaddress$>$ \mbox{[}$<$name$>$\mbox{]}} 

Creates a symbol for a function pointer and associates a specific address as a value for that symbol. The first address specified is the address of the symbol, The second address is the address referred to by the pointer. Indirect calls through the function pointer will be converted to direct calls to the referred address. A symbol name can be provided, otherwise a default one is generated.
