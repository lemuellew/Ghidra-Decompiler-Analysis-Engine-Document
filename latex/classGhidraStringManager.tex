\doxysubsection{Detailed Description}
Implementation of the \mbox{\hyperlink{classStringManager}{String\+Manager}} that queries through the ghidra client. 

This acts as a front end to Ghidra\textquotesingle{}s string formats and encodings. The client translates any type of string into a U\+T\+F8 representation, and this class stores it for final presentation. Escaping the U\+T\+F8 string is left up to the \mbox{\hyperlink{classPrintLanguage}{Print\+Language}}.
