\doxysubsection{Detailed Description}
Transform based on \mbox{\hyperlink{classVarnode}{Varnode}} properties, such as {\itshape read-\/only} and {\itshape volatile}. 

This performs various transforms that are based on \mbox{\hyperlink{classVarnode}{Varnode}} properties.
\begin{DoxyItemize}
\item Read-\/only Varnodes are converted to the underlying constant
\item Volatile Varnodes are converted read/write functions
\item Varnodes whose values are not consumed are replaced with constant 0 Varnodes 
\end{DoxyItemize}
