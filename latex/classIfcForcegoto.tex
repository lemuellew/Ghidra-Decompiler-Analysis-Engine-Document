\doxysubsection{Detailed Description}
Force a branch to be an unstructured {\bfseries{goto\+:}} {\ttfamily force goto $<$branchaddr$>$ $<$targetaddr$>$} 

Create an override that forces the decompiler to treat the specified branch as unstructured. The branch will be modeled as a {\bfseries{goto}} statement. The branch is specified by first providing the address of the branching instruction, then the destination address.
