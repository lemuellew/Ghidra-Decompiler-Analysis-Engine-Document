\doxysubsection{Detailed Description}
Prepare for data-\/flow analysis of function parameters, when recovery isn\textquotesingle{}t required. 

If the \char`\"{}protorecovery\char`\"{} action group is not enabled, this \mbox{\hyperlink{classAction}{Action}} probably should be. It sets up only the potential sub-\/function outputs (not the inputs) otherwise local uses of the output registers may be incorrectly heritaged, screwing up the local analysis (i.\+e. for jump-\/tables) even though we don\textquotesingle{}t care about the function inputs.
