\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+recordtype$>$\newline
class rangemap$<$ \+\_\+recordtype $>$}

An interval map container. 

A container for records occupying (possibly overlapping) intervals. I.\+e. a map from a linear ordered domain to (multiple) records. The {\bfseries{recordtype}} is the main object in the container, it must support\+:
\begin{DoxyItemize}
\item recordtype(inittype,linetype,linetype) a constructor taking 3 parameters
\item get\+First() beginning of range
\item get\+Last() end of range (inclusive)
\item get\+Subsort() retrieve the subsorttype object (see below)
\end{DoxyItemize}

The {\bfseries{recordtype}} must define data-\/types\+:
\begin{DoxyItemize}
\item linetype
\item subsorttype
\item inittype
\end{DoxyItemize}

{\bfseries{linetype}} is the data-\/type of elements in the linear domain. It must support\+:
\begin{DoxyItemize}
\item $<$,$<$= Comparisons
\item ==,!= Equality
\item + $<$constant$>$ Addition of integers
\item -\/ $<$constant$>$ Subtraction of integers
\end{DoxyItemize}

{\bfseries{subsorttype}} describes how overlapping intervals can be sub-\/sorted. It must support\+:
\begin{DoxyItemize}
\item $<$
\item subsorttype({\bfseries{false}}) constructor with {\bfseries{false}} produces a minimal value
\item subsorttype({\bfseries{true}}) constructor with {\bfseries{true}} produces a maximal value
\item copy constructor
\end{DoxyItemize}

{\bfseries{inittype}} is extra initialization data for the {\bfseries{recordtype}} 

The main interval map is implemented as a {\itshape multiset} of disjoint sub-\/ranges mapping to the {\bfseries{recordtype}} objects. After deduping the sub-\/ranges form the common refinement of all the possibly overlapping {\bfseries{recordtype}} ranges. A sub-\/range is duplicated for each distinct {\bfseries{recordtype}} that overlaps that sub-\/range. The sub-\/range multiset is updated with every insertion or deletion of {\bfseries{recordtype}} objects into the container, which may insert new or delete existing boundary points separating the disjoint subranges.
