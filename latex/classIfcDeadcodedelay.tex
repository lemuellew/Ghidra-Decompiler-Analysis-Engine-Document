\doxysubsection{Detailed Description}
Change when dead code elimination starts\+: {\ttfamily deadcode delay $<$name$>$ $<$delay$>$} 

An address space is selected by name, along with a pass number. Dead code elimination for Varnodes in that address space is changed to start during that pass. If there is a {\itshape current} function, the delay is altered only for that function, otherwise the delay is set globally for all functions.
