\doxysubsection{Detailed Description}
An in-\/memory implementation of the \mbox{\hyperlink{classCommentDatabase}{Comment\+Database}} A\+PI. 

All \mbox{\hyperlink{classComment}{Comment}} objects are held in memory in a sorted container. This can be used as stand-\/alone database of comments, or it can act as a cache for some other container.
