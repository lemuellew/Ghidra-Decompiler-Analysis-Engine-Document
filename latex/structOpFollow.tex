\doxysubsection{Detailed Description}
A simple node used to dynamically define a sequence of operations. 

This should be deprecated in favor of \mbox{\hyperlink{classExecutablePcode}{Executable\+Pcode}} objects. This class holds a single operation (within a sequence). It acts on the output of the previous operation with an optional constant value as the second input.
