\doxysubsection{Detailed Description}
A range of values attached to a \mbox{\hyperlink{classVarnode}{Varnode}} within a data-\/flow subsystem. 

This class acts as both the set of values for the \mbox{\hyperlink{classVarnode}{Varnode}} and as a node in a sub-\/graph overlaying the full data-\/flow of the function containing the \mbox{\hyperlink{classVarnode}{Varnode}}. The values are stored in the \mbox{\hyperlink{classCircleRange}{Circle\+Range}} field and can be interpreted either as absolute values (if {\bfseries{type\+Code}} is 0) or as values relative to a stack pointer or some other register (if {\bfseries{type\+Code}} is non-\/zero).
