\doxysubsection{Detailed Description}
A parameter with a formal backing \mbox{\hyperlink{classSymbol}{Symbol}}. 

Input parameters generally have a symbol associated with them. This class holds a reference to the \mbox{\hyperlink{classSymbol}{Symbol}} object and pulls the relevant parameter information off of it.
