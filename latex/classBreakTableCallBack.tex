\doxysubsection{Detailed Description}
A basic instantiation of a breakpoint table. 

This object allows breakpoints to registered in the table via either
\begin{DoxyItemize}
\item \mbox{\hyperlink{classBreakTableCallBack_a7c2ea1b2996ea556188e1c09a8f6ec1f}{register\+Pcode\+Callback()}} or = \mbox{\hyperlink{classBreakTableCallBack_a22f1cb661549be082a71edcca5eb7f5f}{register\+Address\+Callback()}}
\end{DoxyItemize}

Breakpoints are stored in map containers, and the core \mbox{\hyperlink{classBreakTable}{Break\+Table}} methods are implemented to search in these containers
