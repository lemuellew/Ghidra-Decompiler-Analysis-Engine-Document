\doxysubsection{Detailed Description}
Simplify predication constructions involving the I\+N\+T\+\_\+\+OR operator. 

In this form of predication, two variables are set based on a condition and then O\+Red together. Both variables may be set to zero, or to some other value, based on the condition and the zero values are such that at least one of the variables is zero. 
\begin{DoxyCode}{0}
\DoxyCodeLine{tmp1 = cond ? val1 : 0;}
\DoxyCodeLine{tmp2 = cond ?  0 : val2;}
\DoxyCodeLine{result = tmp1 | tmp2;}
\end{DoxyCode}
 The \mbox{\hyperlink{classRuleOrPredicate}{Rule\+Or\+Predicate}} simplifies this to 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordflow}{if} (cond) result = val1; \textcolor{keywordflow}{else} result = val2;}
\end{DoxyCode}
 or to be precise 
\begin{DoxyCode}{0}
\DoxyCodeLine{newtmp = val1 ? val2;           \textcolor{comment}{// Using a new MULTIEQUAL}}
\DoxyCodeLine{result = newtmp;}
\end{DoxyCode}
 In an alternate form we have 
\begin{DoxyCode}{0}
\DoxyCodeLine{tmp1 = (val2 == 0) ? val1 : 0}
\DoxyCodeLine{result = tmp1 | val2;}
\end{DoxyCode}
 again, one of val1 or val2 must be zero, so this gets replaced with 
\begin{DoxyCode}{0}
\DoxyCodeLine{tmp1 = val1 ? val2}
\DoxyCodeLine{result = tmp1}
\end{DoxyCode}
