\doxysubsection{Detailed Description}
Make a specific function current\+: {\ttfamily load function $<$functionname$>$} 

The name must be a fully qualified symbol with \char`\"{}\+::\char`\"{} separating namespaces. If the symbol represents a function, that function becomes {\itshape current} for the console. If there are bytes for the function, raw p-\/code and control-\/flow are calculated.
