\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classAddress}{Address}} range associated with the symbol \mbox{\hyperlink{classScope}{Scope}} that owns it. 

As part of a rangemap, this forms a map from addresses to {\itshape namespace} Scopes so that the decompiler can quickly find the {\itshape namespace} \mbox{\hyperlink{classScope}{Scope}} that holds the \mbox{\hyperlink{classSymbol}{Symbol}} it sees accessed.
