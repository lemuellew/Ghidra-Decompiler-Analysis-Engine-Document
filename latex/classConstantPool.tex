\doxysubsection{Detailed Description}
An interface to the pool of {\bfseries{constant}} objects for byte-\/code languages. 

This is an abstract base class that acts as a container for C\+Pool\+Records. A {\itshape reference} (1 or more integer constants) maps to an individual \mbox{\hyperlink{classCPoolRecord}{C\+Pool\+Record}}. A \mbox{\hyperlink{classCPoolRecord}{C\+Pool\+Record}} object can be queried for using \mbox{\hyperlink{classConstantPool_ac17df35ff1376311f62912d29bf26cad}{get\+Record()}}, and a new object can be added with \mbox{\hyperlink{classConstantPool_a004b94bca379a22b84d11b7299eabdd8}{put\+Record()}}. Internally, the actual \mbox{\hyperlink{classCPoolRecord}{C\+Pool\+Record}} object is produced by create\+Record().
