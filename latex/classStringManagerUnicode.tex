\doxysubsection{Detailed Description}
An implementation of \mbox{\hyperlink{classStringManager}{String\+Manager}} that understands terminated unicode strings. 

This class understands U\+T\+F8, U\+T\+F16, and U\+T\+F32 encodings. It reports a string if its sees a valid encoding that is null terminated.
