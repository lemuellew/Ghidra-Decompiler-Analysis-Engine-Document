\doxysubsection{Detailed Description}
Mark a memory range as read-\/only\+: {\ttfamily readonly $<$address+size$>$} 

The memory range provided on the command-\/line is marked as {\itshape read-\/only}, which allows the decompiler to propagate values pulled from the \mbox{\hyperlink{classLoadImage}{Load\+Image}} for the range as constants.
