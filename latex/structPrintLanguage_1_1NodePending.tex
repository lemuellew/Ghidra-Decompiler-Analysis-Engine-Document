\doxysubsection{Detailed Description}
A pending data-\/flow node; waiting to be placed on the reverse polish notation stack. 

This holds an {\itshape implied} \mbox{\hyperlink{classVarnode}{Varnode}} in the data-\/flow graph, which prints as the expression producing the value in the \mbox{\hyperlink{classVarnode}{Varnode}}.
