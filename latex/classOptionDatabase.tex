\doxysubsection{Detailed Description}
A Dispatcher for possible \mbox{\hyperlink{classArchOption}{Arch\+Option}} commands. 

An {\bfseries{option}} {\bfseries{command}} is a specific request by a user to change the configuration options for an \mbox{\hyperlink{classArchitecture}{Architecture}}. This class takes care of dispatching the command to the proper \mbox{\hyperlink{classArchOption}{Arch\+Option}} derived class, which does the work of actually modifying the configuration. The command is issued either through the \mbox{\hyperlink{classOptionDatabase_aaec7650371cbbb2a17a88393347c9ffc}{set()}} method directly, or via an X\+ML tag handed to the \mbox{\hyperlink{classOptionDatabase_afe0ea0f1c7a7cd2591ac159027082149}{restore\+Xml()}} method. The \mbox{\hyperlink{classOptionDatabase_afe0ea0f1c7a7cd2591ac159027082149}{restore\+Xml()}} method expects an $<$optionslist$>$ tag with one or more sub-\/tags. The sub-\/tag names match the registered name of the option and have up to three child tags, $<$param1$>$, $<$param2$>$ and $<$param3$>$, whose content is provided as the optional parameters to command.
