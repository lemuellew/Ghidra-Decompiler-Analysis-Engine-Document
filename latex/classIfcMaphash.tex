\doxysubsection{Detailed Description}
Add a dynamic symbol to the current function\+: {\ttfamily map hash $<$address$>$ $<$hash$>$ $<$typedeclaration$>$} 

The command only creates local variables for the current function. The name and data-\/type are taken from a C syntax type declaration. The symbol is not associated with a particular storage address but with a specific \mbox{\hyperlink{classVarnode}{Varnode}} in the data-\/flow, specified by a code address and hash of the local data-\/flow structure.
