\doxysubsection{Detailed Description}
A S\+AX interface implementation for constructing an in-\/memory D\+OM model. 

This implementation builds a D\+OM model of the X\+ML stream being parsed, creating an \mbox{\hyperlink{classElement}{Element}} object for each X\+ML element tag in the stream. This handler is initialized with a root \mbox{\hyperlink{classElement}{Element}} object, which after parsing is complete will own all parsed elements.
