\doxysubsection{Detailed Description}
Simple unit test class. 

The macro T\+E\+ST instantiates this object with a name and function pointer. The static \mbox{\hyperlink{structUnitTest_ac3027b4d9fd45f681b9a896721227117}{run()}} method calls all the function pointers of all instantiated objects.
