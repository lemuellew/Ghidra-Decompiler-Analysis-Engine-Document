\doxysubsection{Detailed Description}
A loop structure where the condition is checked at the bottom. 

This has exactly one component with two outgoing edges\+: one edge flows to itself, the other flows to the exit block. The \mbox{\hyperlink{classBlockDoWhile}{Block\+Do\+While}} instance has exactly one outgoing edge.
