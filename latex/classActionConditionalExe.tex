\doxysubsection{Detailed Description}
Search for and remove various forms of redundant C\+B\+R\+A\+N\+CH operations. 

This action wraps the analysis performed by \mbox{\hyperlink{classConditionalExecution}{Conditional\+Execution}} to simplify control-\/flow that repeatedly branches on the same (or slightly modified) boolean expression.
