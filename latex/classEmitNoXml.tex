\doxysubsection{Detailed Description}
A trivial emitter that outputs syntax straight to the stream. 

This emitter does neither pretty printing nor X\+ML markup. It dumps any tokens straight to the final output stream. It can be used as the low-\/level back-\/end for \mbox{\hyperlink{classEmitPrettyPrint}{Emit\+Pretty\+Print}}.
