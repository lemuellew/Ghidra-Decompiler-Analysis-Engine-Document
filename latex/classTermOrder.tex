\doxysubsection{Detailed Description}
A class for ordering \mbox{\hyperlink{classVarnode}{Varnode}} terms in an additive expression. 

Given the final \mbox{\hyperlink{classPcodeOp}{Pcode\+Op}} in a data-\/flow expression that sums 2 or more \mbox{\hyperlink{classVarnode}{Varnode}} {\itshape terms}, this class collects all the terms then allows sorting of the terms to facilitate constant collapse and factoring simplifications.
