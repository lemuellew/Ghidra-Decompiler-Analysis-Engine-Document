Set the value of a register\+: {\ttfamily set track $<$name$>$ $<$value$>$ \mbox{[}$<$startaddress$>$ $<$endaddress$>$\mbox{]}} 

The value for the register is picked up by the decompiler for functions in the tracked range. The register is specified by name. A specific range can be provided, otherwise the value is treated as a default.
