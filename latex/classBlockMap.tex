\doxysubsection{Detailed Description}
Helper class for resolving cross-\/references while deserializing \mbox{\hyperlink{classBlockGraph}{Block\+Graph}} objects. 

\mbox{\hyperlink{classFlowBlock}{Flow\+Block}} objects are serialized with their associated {\bfseries{index}} value and edges are serialized with the indices of the \mbox{\hyperlink{classFlowBlock}{Flow\+Block}} end-\/points. During deserialization, this class maintains a list of \mbox{\hyperlink{classFlowBlock}{Flow\+Block}} objects sorted by index and then looks up the \mbox{\hyperlink{classFlowBlock}{Flow\+Block}} matching a given index as edges specify them.
