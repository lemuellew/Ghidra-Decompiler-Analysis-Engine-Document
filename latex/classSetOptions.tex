\doxysubsection{Detailed Description}
Command to {\bfseries{toggle}} {\bfseries{options}} within the decompiler. 

The decompiler supports configuration of a variety of named options that affect everything from how code is transformed to how it is displayed (See \mbox{\hyperlink{classArchOption}{Arch\+Option}}). The command expects 2 string parameters\+: the encoded integer id of the program, and an X\+ML document containing an $<$optionslist$>$ tag. The $<$optionslist$>$ tag contains one child tag for each option to be configured. The command returns a single character message, \textquotesingle{}t\textquotesingle{} or \textquotesingle{}f\textquotesingle{}, indicating whether the configuration succeeded.
