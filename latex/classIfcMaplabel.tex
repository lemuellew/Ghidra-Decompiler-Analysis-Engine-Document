\doxysubsection{Detailed Description}
Create a code label\+: {\ttfamily map label $<$name$>$ $<$address$>$} 

Label a specific code address. This creates a \mbox{\hyperlink{classLabSymbol}{Lab\+Symbol}} which is usually an internal control-\/flow target. The symbol is local to the {\itshape current} function if it exists, otherwise the symbol is added to the global scope.
