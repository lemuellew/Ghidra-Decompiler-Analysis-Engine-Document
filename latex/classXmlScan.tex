\doxysubsection{Detailed Description}
The X\+ML character scanner. 

Tokenize a byte stream suitably for the main X\+ML parser. The scanner expects an A\+S\+C\+II or U\+T\+F-\/8 encoding. Characters is X\+ML tag and attribute names are restricted to A\+S\+C\+II \char`\"{}letters\char`\"{}, but extended U\+T\+F-\/8 characters can be used in any other character data\+: attribute values, content, comments. ~\newline
