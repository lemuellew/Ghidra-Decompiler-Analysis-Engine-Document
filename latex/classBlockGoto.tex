\doxysubsection{Detailed Description}
A block that terminates with an unstructured (goto) branch to another block. 

The {\itshape goto} must be an unconditional branch. The instance keeps track of the target block and will emit the branch as some form of formal branch statement (goto, break, continue). From the point of view of control-\/flow structuring, this block has {\itshape no} output edges. The algorithm handles edges it can\textquotesingle{}t structure by encapsulating it in the \mbox{\hyperlink{classBlockGoto}{Block\+Goto}} class and otherwise removing the edge from the structured view of the graph.
