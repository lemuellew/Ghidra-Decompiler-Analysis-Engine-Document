\doxysubsection{Detailed Description}
Set a breakpoint when a \mbox{\hyperlink{classRule}{Rule}} or \mbox{\hyperlink{classAction}{Action}} executes\+: {\ttfamily break action $<$actionname$>$} 

The break point can be on either an \mbox{\hyperlink{classAction}{Action}} or \mbox{\hyperlink{classRule}{Rule}}. The name can specify partial path information to distinguish the Action/\+Rule. The breakpoint causes the decompilation process to stop and return control to the console immediately {\itshape after} the \mbox{\hyperlink{classAction}{Action}} or \mbox{\hyperlink{classRule}{Rule}} has executed, but only if there was an active transformation to the function.
