\doxysubsection{Detailed Description}
A container for disassembly context used by the S\+L\+E\+I\+GH engine. 

This acts as a factor for the \mbox{\hyperlink{classParserContext}{Parser\+Context}} objects which are used to disassemble a single instruction. These all share a \mbox{\hyperlink{classContextCache}{Context\+Cache}} which is a front end for accessing the \mbox{\hyperlink{classContextDatabase}{Context\+Database}} and resolving context variables from the S\+L\+E\+I\+GH spec. \mbox{\hyperlink{classParserContext}{Parser\+Context}} objects are stored in a hash-\/table keyed by the address of the instruction.
