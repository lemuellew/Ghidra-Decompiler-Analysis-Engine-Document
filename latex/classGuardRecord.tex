\doxysubsection{Detailed Description}
A (putative) switch variable \mbox{\hyperlink{classVarnode}{Varnode}} and a constraint imposed by a C\+B\+R\+A\+N\+CH. 

The record constrains a specific \mbox{\hyperlink{classVarnode}{Varnode}}. If the associated C\+B\+R\+A\+N\+CH is followed along the path that reaches the switch\textquotesingle{}s B\+R\+A\+N\+C\+H\+I\+ND, then we have an explicit description of the possible values the \mbox{\hyperlink{classVarnode}{Varnode}} can hold.
