\doxysubsection{Detailed Description}
An injection payload that uses a Ghidra client to generate the p-\/code ops. 

This acts as a placeholder for the actual details of the payload. When the \mbox{\hyperlink{classInjectPayloadGhidra_ad6f4913cfcdd2eaf229a45150802250c}{inject()}} method is invoked, the context is wrapped as X\+ML and sent to the Ghidra client, which returns the actual p-\/code to inject.
