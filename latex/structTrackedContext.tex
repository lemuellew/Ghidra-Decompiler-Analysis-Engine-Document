\doxysubsection{Detailed Description}
A tracked register (\mbox{\hyperlink{classVarnode}{Varnode}}) and the value it contains. 

This is the object returned when querying for tracked registers, via \mbox{\hyperlink{classContextDatabase_a6ea1e4b793286c0b1ecbafcc58422339}{Context\+Database\+::get\+Tracked\+Set()}}. It holds the storage details of the register and the actual value it holds at the point of the query.
