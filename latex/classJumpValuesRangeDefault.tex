\doxysubsection{Detailed Description}
A jump-\/table starting range with two possible execution paths. 

This extends the basic \mbox{\hyperlink{classJumpValuesRange}{Jump\+Values\+Range}} having a single entry switch variable and adds a second entry point that takes only a single value. This value comes last in the iteration.
