\doxysubsection{Detailed Description}
Attach a named symbol to a specific \mbox{\hyperlink{classVarnode}{Varnode}}\+: {\ttfamily name varnode $<$varnode$>$ $<$name$>$} 

A new local symbol is created for the {\itshape current} function, and is attached to the specified \mbox{\hyperlink{classVarnode}{Varnode}}. The {\itshape current} function must be decompiled again to see the effects. The new symbol is {\itshape name-\/locked} with the specified name, but the data-\/type of the symbol is allowed to float.
