\doxysubsection{Detailed Description}
Class for freezing value sets at a specific iteration (to accelerate convergence) 

The value sets don\textquotesingle{}t reach a true stable state but instead lock in a description of the first few values that {\itshape reach} a given \mbox{\hyperlink{classVarnode}{Varnode}}. The \mbox{\hyperlink{classValueSetSolver}{Value\+Set\+Solver}} does normal iteration, but individual Value\+Sets {\itshape freeze} after a specific number of iterations (3 by default), instead of growing to a true stable state. This gives evidence of iteration in the underlying code, showing the initial value and frequently the step size.
