\doxysubsection{Detailed Description}
Create a new function\+: {\ttfamily map function $<$address$>$ \mbox{[}$<$functionname$>$\mbox{]} \mbox{[}nocode\mbox{]}} 

Create a new function symbol at the provided address. A symbol name can be provided, otherwise a default name is selected. The new function becomes {\itshape current} for the console. The provided address gives the entry point for the function. Unless the final keyword \char`\"{}nocode\char`\"{} is provided, the underlying bytes in the load image are used for any future disassembly or decompilation.
