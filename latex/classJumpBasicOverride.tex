\doxysubsection{Detailed Description}
A basic jump-\/table model incorporating manual override information. 

The list of potential target addresses produced by the B\+R\+A\+N\+C\+H\+I\+ND is not recovered by {\bfseries{this}} model, but must provided explicitly via \mbox{\hyperlink{classJumpBasicOverride_add96a354fac8785292195cb321899544}{set\+Addresses()}}. The model tries to repurpose some of the analysis that \mbox{\hyperlink{classJumpBasic}{Jump\+Basic}} does to recover the switch variable. But it will revert to the trivial model if it can\textquotesingle{}t find a suitable switch variable.
