\doxysubsection{Detailed Description}
Space for storing internal \mbox{\hyperlink{classPcodeOp}{Pcode\+Op}} pointers as addresses. 

It is convenient and efficient to replace the formally encoded branch target addresses with a pointer to the actual \mbox{\hyperlink{classPcodeOp}{Pcode\+Op}} being branched to. This special {\bfseries{iop}} space allows a \mbox{\hyperlink{classPcodeOp}{Pcode\+Op}} pointer to be encoded as an address so it can be stored as part of an input varnode, in place of the target address, in a {\itshape branching} operation. The pointer is encoded as an offset within the {\bfseries{fspec}} space.
