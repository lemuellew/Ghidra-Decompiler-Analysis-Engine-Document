\doxysubsection{Detailed Description}
Perform \mbox{\hyperlink{classSubvariableFlow}{Subvariable\+Flow}} analysis triggered by testing of a single bit. 

Given a comparison (I\+N\+T\+\_\+\+E\+Q\+U\+AL or I\+N\+T\+\_\+\+N\+O\+T\+E\+E\+Q\+U\+A\+L\+\_\+ to a constant, check that input has only 1 bit that can possibly be non-\/zero and that the constant is testing this. This then triggers the full \mbox{\hyperlink{classSubvariableFlow}{Subvariable\+Flow}} analysis.
