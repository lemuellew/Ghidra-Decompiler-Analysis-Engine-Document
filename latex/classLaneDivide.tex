\doxysubsection{Detailed Description}
Class for splitting data-\/flow on {\itshape laned} registers. 

From a root \mbox{\hyperlink{classVarnode}{Varnode}} and a description of its {\itshape lanes}, trace data-\/flow as far as possible through the function, propagating each lane, using the \mbox{\hyperlink{classLaneDivide_ae5f598f2a49cc5813d759441125cc4b3}{do\+Trace()}} method.
