\doxysubsection{Detailed Description}
An interface to a container of comments. 

Comments can be added (and removed) from a database, keying on the function and address the \mbox{\hyperlink{classComment}{Comment}} is attached to. The interface can generate a {\itshape begin} and {\itshape end} iterator covering all \mbox{\hyperlink{classComment}{Comment}} objects for a single function.
