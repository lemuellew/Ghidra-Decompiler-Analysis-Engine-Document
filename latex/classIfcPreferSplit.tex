\doxysubsection{Detailed Description}
Mark a storage location to be split\+: {\ttfamily prefersplit $<$address+size$>$ $<$splitsize$>$} 

The storage location is marked for splitting in any future decompilation. During decompilation, any \mbox{\hyperlink{classVarnode}{Varnode}} matching the storage location on the command-\/line will be generally split into two pieces, where the final command-\/line parameter indicates the number of bytes in the first piece. A \mbox{\hyperlink{classVarnode}{Varnode}} is split only if operations involving it can also be split. See \mbox{\hyperlink{classPreferSplitManager}{Prefer\+Split\+Manager}}.
