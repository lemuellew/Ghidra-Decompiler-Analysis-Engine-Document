\doxysubsection{Detailed Description}
Print change to stack pointer for called function\+: {\ttfamily print extrapop \mbox{[}$<$functionname$>$\mbox{]}} 

For the selected function, the extra amount each called function changes the stack pointer (over popping the return value) is printed to console. The function is selected by name, or if no name is given, the {\itshape current} function is selected.
