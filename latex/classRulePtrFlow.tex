\doxysubsection{Detailed Description}
Mark \mbox{\hyperlink{classVarnode}{Varnode}} and \mbox{\hyperlink{classPcodeOp}{Pcode\+Op}} objects that are carrying or operating on pointers. 

This is used on architectures where the data-\/flow for pointer values needs to be truncated. This marks the places where the truncation needs to happen. Then the \mbox{\hyperlink{classSubvariableFlow}{Subvariable\+Flow}} actions do the actual truncation.
