\doxysubsection{Detailed Description}
Class for calculating \char`\"{}goodness of fit\char`\"{} of parameter trials against a prototype model. 

The class is instantiated with a prototype model (\mbox{\hyperlink{classProtoModel}{Proto\+Model}}). A set of \mbox{\hyperlink{classVarnode}{Varnode}} parameter trials are registered by calling \mbox{\hyperlink{classScoreProtoModel_aac1568db9b3ebba5747ad8eba5f6438f}{add\+Parameter()}} for each trial. Then calling \mbox{\hyperlink{classScoreProtoModel_aae328b10f50c1f6526f503e0064d4749}{do\+Score()}} computes a score that evaluates how well the set of registered trials fit the prototype model. A lower score indicates a better fit.
