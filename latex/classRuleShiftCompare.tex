\doxysubsection{Detailed Description}
Transform shifts in comparisons\+: {\ttfamily V $>$$>$ c == d =$>$ V == (d $<$$<$ c)} 

Similarly\+: {\ttfamily V $<$$<$ c == d =$>$ V \& mask == (d $>$$>$ c)}

The rule works on both I\+N\+T\+\_\+\+E\+Q\+U\+AL and I\+N\+T\+\_\+\+N\+O\+T\+E\+Q\+U\+AL.
