\doxysubsection{Detailed Description}
An operation that reads from volatile memory. 

This C\+A\+L\+L\+O\+T\+H\+ER p-\/code operation takes as its input parameter, after the constant id, a reference \mbox{\hyperlink{classVarnode}{Varnode}} to the memory being read. The output returned by this operation is the actual value read from memory.
