\doxysubsection{Detailed Description}
A collection of parameter descriptions backed by \mbox{\hyperlink{classSymbol}{Symbol}} information. 

Input parameters are determined by symbols a function \mbox{\hyperlink{classScope}{Scope}} (category 0). Information about the return-\/value is stored internally. \mbox{\hyperlink{classProtoParameter}{Proto\+Parameter}} objects are constructed on the fly as requested and cached.
