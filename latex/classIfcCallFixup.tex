\doxysubsection{Detailed Description}
Add a new call fix-\/up to the program\+: {\ttfamily fixup call ...} 

Create a new call fixup-\/up for the architecture/program, suitable for replacing called functions. The fix-\/up is specified as a function-\/style declarator, which also provides the formal name of the fix-\/up. A \char`\"{}void\char`\"{} return-\/type and empty parameter list must be given.
