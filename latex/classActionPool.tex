\doxysubsection{Detailed Description}
A pool of Rules that apply simultaneously. 

This class groups together a set of Rules as a formal \mbox{\hyperlink{classAction}{Action}}. Rules are given an opportunity to apply to every \mbox{\hyperlink{classPcodeOp}{Pcode\+Op}} in a function. Usually rule\+\_\+repeatapply is enabled for this action, which causes all Rules to apply repeatedly until no \mbox{\hyperlink{classRule}{Rule}} can make an additional change.
