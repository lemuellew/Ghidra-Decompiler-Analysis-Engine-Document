\doxysubsection{Detailed Description}
Class for splitting up Varnodes that hold 2 logical variables. 

Starting from a {\itshape root} \mbox{\hyperlink{classVarnode}{Varnode}} provided to the constructor, {\bfseries{this}} class looks for data-\/flow that consistently holds 2 logical values in a single \mbox{\hyperlink{classVarnode}{Varnode}}. If \mbox{\hyperlink{classSplitFlow_a45e3612033f7082f63af36a551f64917}{do\+Trace()}} returns {\bfseries{true}}, a consistent view has been created and invoking \mbox{\hyperlink{classTransformManager_a2dcee2b69511dfb160421566ad1e9ce8}{apply()}} will split all Varnodes and Pcode\+Ops involved in the data-\/flow into their logical pieces.
