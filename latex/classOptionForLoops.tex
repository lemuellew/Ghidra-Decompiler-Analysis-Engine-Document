\doxysubsection{Detailed Description}
Toggle whether the decompiler attempts to recover {\itshape for-\/loop} variables. 

Setting the first parameter to \char`\"{}on\char`\"{} causes the decompiler to search for a suitable loop variable controlling iteration of a {\itshape while-\/do} block. The {\itshape for-\/loop} displays the following on a single line\+:
\begin{DoxyItemize}
\item loop variable initializer (optional)
\item loop condition
\item loop variable incrementer 
\end{DoxyItemize}
