\doxysubsection{Detailed Description}
Class for describing a relative p-\/code branch destination. 

An intra-\/instruction p-\/code branch takes a {\itshape relative} operand. The actual value produced during p-\/code generation is calculated at the last second using {\bfseries{this}}. It stores the index of the B\+R\+A\+N\+CH instruction and a reference to its destination operand. This initially holds a reference to a destination {\itshape label} symbol, but is later updated with the final relative value.
