\doxysubsection{Detailed Description}
Command to {\bfseries{release}} all resources associated with a Program (executable) in the decompiler. 

The command frees the \mbox{\hyperlink{classArchitectureGhidra}{Architecture\+Ghidra}} object (recursively affecting all resources) associated with the program. A {\itshape termination} meta-\/command is issued for this process. The command expects a single string parameter encoding the id of the program.
