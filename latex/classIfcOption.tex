\doxysubsection{Detailed Description}
Adjust a decompiler option\+: {\ttfamily option $<$optionname$>$ \mbox{[}$<$param1$>$\mbox{]} \mbox{[}$<$param2$>$\mbox{]} \mbox{[}$<$param3$>$\mbox{]}} 

Passes command-\/line parameters to an \mbox{\hyperlink{classArchOption}{Arch\+Option}} object registered with the current architecture\textquotesingle{}s \mbox{\hyperlink{classOptionDatabase}{Option\+Database}}. Options are looked up by name and can be configure with up to 3 parameters. Options generally report success or failure back to the console.
