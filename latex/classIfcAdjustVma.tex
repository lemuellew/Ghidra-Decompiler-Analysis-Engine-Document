\doxysubsection{Detailed Description}
Change the base address of the load image\+: {\ttfamily adjust vma 0xabcd0123} 

The provided parameter is added to the current base address of the image. This only affects the address of bytes in the image and so should be done before functions and other symbols are layed down.
