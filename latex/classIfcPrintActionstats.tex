\doxysubsection{Detailed Description}
Print transform statistics for the decompiler engine\+: {\ttfamily print actionstats} 

Counts for each \mbox{\hyperlink{classAction}{Action}} and \mbox{\hyperlink{classRule}{Rule}} are displayed; showing the number of attempts, both successful and not, that were made to apply each one. Counts can accumulate over multiple decompilations.
