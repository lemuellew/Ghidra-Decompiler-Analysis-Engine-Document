\doxysubsection{Detailed Description}
Set a break point at the start of an \mbox{\hyperlink{classAction}{Action}}\+: {\ttfamily break start $<$actionname$>$} 

The break point can be on either an \mbox{\hyperlink{classAction}{Action}} or a \mbox{\hyperlink{classRule}{Rule}}. The name can specify partial path information to distinguish the Action/\+Rule. The breakpoint causes the decompilation process to stop and return control to the console just before the Action/\+Rule would have executed.
