\doxysubsection{Detailed Description}
Provide an overriding jump-\/table for an indirect branch\+: {\ttfamily override jumptable ...} 

The command expects the address of an indirect branch in the {\itshape current} function, followed by the keyword {\bfseries{table}} then a list of possible target addresses of the branch. 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{override} jumptable r0x1000 table r0x1020 r0x1030 r0x1043 ...}
\end{DoxyCode}
 The command can optionally take the keyword {\bfseries{startval}} followed by an integer indicating the value taken by the {\itshape normalized} switch variable that produces the first address in the table. 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{override} jumptable startval 10 table r0x1020 r0x1030 ...}
\end{DoxyCode}
