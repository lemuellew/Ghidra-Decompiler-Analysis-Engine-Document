\doxysubsection{Detailed Description}
A trivial jump-\/table model, where the B\+R\+A\+N\+C\+H\+I\+ND input \mbox{\hyperlink{classVarnode}{Varnode}} is the switch variable. 

This class treats the input \mbox{\hyperlink{classVarnode}{Varnode}} to the B\+R\+A\+N\+C\+H\+I\+ND as the switch variable, and recovers its possible values from the existing block structure. This is used when the flow following fork recovers destination addresses, but the switch normalization action is unable to recover the model.
