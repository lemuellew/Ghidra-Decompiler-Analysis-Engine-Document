\doxysubsection{Detailed Description}
An infinite loop structure. 

This has exactly one component with one outgoing edge that flows into itself. The \mbox{\hyperlink{classBlockInfLoop}{Block\+Inf\+Loop}} instance has zero outgoing edges.
