\doxysubsection{Detailed Description}
Description of logical lanes within a {\bfseries{big}} \mbox{\hyperlink{classVarnode}{Varnode}}. 

A {\bfseries{lane}} is a byte offset and size within a \mbox{\hyperlink{classVarnode}{Varnode}}. Lanes within a \mbox{\hyperlink{classVarnode}{Varnode}} are disjoint. In general, we expect a \mbox{\hyperlink{classVarnode}{Varnode}} to be tiled with lanes all of the same size, but the A\+PI allows for possibly non-\/uniform lanes.
