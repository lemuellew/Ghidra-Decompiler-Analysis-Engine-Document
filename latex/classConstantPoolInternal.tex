\doxysubsection{Detailed Description}
An implementation of the \mbox{\hyperlink{classConstantPool}{Constant\+Pool}} interface storing records internally in R\+AM. 

The \mbox{\hyperlink{classCPoolRecord}{C\+Pool\+Record}} objects are held directly in a map container. This class can be used as a stand-\/alone \mbox{\hyperlink{classConstantPool}{Constant\+Pool}} that holds all its records in R\+AM. Or, it can act as a local \mbox{\hyperlink{classCPoolRecord}{C\+Pool\+Record}} cache for some other implementation.
