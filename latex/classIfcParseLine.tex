\doxysubsection{Detailed Description}
Parse a line of C syntax\+: {\ttfamily parse line ...} 

The line can contain a declaration either a data-\/type or a function prototype\+:
\begin{DoxyItemize}
\item {\ttfamily parse line typedef int4 $\ast$specialint;}
\item {\ttfamily parse line struct mystruct \{ int4 a; int4 b; \}}
\item {\ttfamily parse line extern void myfunc(int4 a,int4 b);}
\end{DoxyItemize}

Data-\/types go straight into the program. For a prototype, the function symbol must already exist.
