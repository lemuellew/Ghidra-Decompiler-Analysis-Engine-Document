\doxysubsection{Detailed Description}
Special \mbox{\hyperlink{classDatatype}{Datatype}} object used to describe pointers that index into the symbol table. 

A \mbox{\hyperlink{classTypeSpacebase}{Type\+Spacebase}} treats a specific \mbox{\hyperlink{classAddrSpace}{Addr\+Space}} as \char`\"{}structure\char`\"{} that will get indexed in to. This facilitates type propagation from local symbols into the stack space and from global symbols into the R\+AM space.
