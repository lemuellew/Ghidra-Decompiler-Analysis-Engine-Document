\doxysubsection{Detailed Description}
Helper class associating a \mbox{\hyperlink{classVarnode}{Varnode}} with the block where it is defined. 

This class explicitly stores a \mbox{\hyperlink{classVarnode}{Varnode}} with the index of the \mbox{\hyperlink{classBlockBasic}{Block\+Basic}} that defines it. If a \mbox{\hyperlink{classVarnode}{Varnode}} does not have a defining \mbox{\hyperlink{classPcodeOp}{Pcode\+Op}} it is assigned an index of 0. This facilitates quicker sorting of Varnodes based on their defining block.
