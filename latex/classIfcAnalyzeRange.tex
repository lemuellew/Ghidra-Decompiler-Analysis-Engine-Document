\doxysubsection{Detailed Description}
Run value-\/set analysis on the {\itshape current} function\+: {\ttfamily analyze range full$\vert$partial $<$varnode$>$} 

The analysis targets a single varnode as specified on the command-\/line and is based on the existing data-\/flow graph for the current function. The possible values that can reach the varnode at its point of definition, and at any point it is involved in a L\+O\+AD or S\+T\+O\+RE, are displayed. The keywords {\bfseries{full}} and {\bfseries{partial}} choose whether the value-\/set analysis uses full or partial widening.
