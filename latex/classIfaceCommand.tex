\doxysubsection{Detailed Description}
A command that can be executed from the command line. 

The command has data associated with it (via \mbox{\hyperlink{classIfaceCommand_ad0a0ba80d392ef1346c43283a8c3ca90}{set\+Data()}}) and is executed via the \mbox{\hyperlink{classIfaceCommand_af10e29cee2c8e419de6efe9e680ad201}{execute()}} method. The command can get additional parameters from the command line by reading the input stream passed to it. The command is associated with a specific sequence of words (tokens) that should appear at the start of the command line.
