\doxysubsection{Detailed Description}
The list of groups defining a {\itshape root} \mbox{\hyperlink{classAction}{Action}}. 

Any \mbox{\hyperlink{classRule}{Rule}} or {\itshape leaf} \mbox{\hyperlink{classAction}{Action}} belongs to a {\bfseries{group}}. This class is a {\bfseries{grouplist}} defined by a collection of these {\bfseries{group}} names. The set of \mbox{\hyperlink{classRule}{Rule}} and \mbox{\hyperlink{classAction}{Action}} objects belong to any of the groups in this list together form a {\bfseries{root}} \mbox{\hyperlink{classAction}{Action}}.
