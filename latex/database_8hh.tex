\doxysubsection{Detailed Description}
\mbox{\hyperlink{classSymbol}{Symbol}} and \mbox{\hyperlink{classScope}{Scope}} objects for the decompiler. 

These implement the main symbol table, with support for symbols, local and global scopes, namespaces etc. Search can be by name or the address of the \mbox{\hyperlink{classSymbol}{Symbol}} storage location.
