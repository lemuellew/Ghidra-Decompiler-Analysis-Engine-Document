\doxysubsection{Detailed Description}
Class holding a particular widening strategy for the \mbox{\hyperlink{classValueSetSolver}{Value\+Set\+Solver}} iteration algorithm. 

This obects gets to decide when a value set gets {\itshape frozen} (\mbox{\hyperlink{classWidener_a44e6ae8f9832a564a2e8f9b86fa72d83}{check\+Freeze()}}), meaning the set doesn\textquotesingle{}t change for the remaining iteration steps. It also gets to decide when and by how much value sets get artificially increased in size to accelerate reaching their stable state (\mbox{\hyperlink{classWidener_a5c9bd4db7928361e7277cdbf3d8287c8}{do\+Widening()}}).
